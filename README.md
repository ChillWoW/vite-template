# A fully functional Vite template

## Disclaimer

-   I am aware that this uses tailwind version 3.x and this is intentional
    -   This has been done like this because I have the need to use it currently at only 3.x

## Installation

### 1. Install

```
npm install
```

### 2. Run

```
npm run dev
```

### 3. Build

```
npm run build
```
